# node-react-sse
利用node层转发sse流式输出数据到前端页面

# 步骤
***
	1、准备nodejs >= 18，推荐19.1.0
	2、yarn install 或 npm install
	3、yarn build
	4、node index.js
	5、访问http://localhost:8686
***

# 介绍
***
	前端框架：react+sass+webpack+antd
	后端：nodejs+express+axios
	md插件库：react-markdown+markdown-it	
***
